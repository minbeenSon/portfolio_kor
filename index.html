HOW TO STUDY

다양한 컴퓨터 언어 중 JAVA 라는 언어가 가장 많이 사용된다고 생각되어 강남의 한 학원에 가서 한번 배워보자라는 마음으로 약 2달정도 배웠습니다. JAVA의 여러 문법들과 그것들을 이용해 여러 기능들을 만들어 보았습니다. 하지만 눈에 보이지 않는 기능들에 점점 제 스스로 이걸로 무엇을 만들 수 있을까라는 의구심이 생겼던 것 같습니다.

그래서 일단 눈에 보이는 Front 쪽 공부를 해보자는 마음으로 강의를 찾아보다 생활코딩이라는 무료 강의 사이트를 발견하고 그 곳에서 HTML과 CSS과 JavaScript 의 기초를 배울 수 있었습니다.
그 후 Front 쪽으로 더욱 공부하고 싶다는 생각을 하였고 애플코딩이라는 유료 강의사이트를 통해 조금 더 심화된 HTML,CSS,JavaScript를 공부했고 또 React라는 프레임워크를 공부하였습니다.

최근에는 Front와 Back 모두를 이용해 다뤄 사이트를 만들어 보고 싶다 라는 생각이 들어 JSP, JAVA, MySQL 을 이용해 로그인 기능과 게시판을 만들어 보았습니다.

1. 반응형 웹 / 애니매이션

  • keyframe 을 이용해 버튼의 애니매이션을 만들어 보았고, CSS 3D(flip) 를 이용해 뒤집히는 애니매이션을 만들어 보았습니다.
  • Navbar를 스크롤해도 보일 수 있게 fixed 를 했고, 내용을 보는대 조금이나마 방해을 줄이기 위해, 아래로 스크롤 될 때 사이즈를 작아지게 했고 어둡게 투명도를 주었습니다.
  • Carosel를 라이브러리 사용하지 않고 직접 JavaScript를 사용해 만들어 보았습니다.
2. 로그인 / 회원가입(이메일 인증)
  • MySQL과 연결하여 회원가입과 로그인 기능을 만들었습니다.
  • 로그인 상태에서는 로그아웃만 보이도록 만들었습니다.
  • 회원가입 아이디,비밀번호,이메일에는 정규식을 사용했고 정규식에 부합하지 않은 경우 알림이 뜨게 했습니다.
  • 이메일 인증 기능은 SHA(Secure Hash Algorithm)를 이용해서 보안 기능을 만들어 보았습니다.
  • 글쓰기 기능은 로그인과 이메일 인증 후에만 사용할 수 있도록 만들었습니다.
  • 로그인은 되었지만 이메일 인증이 안되어 있는 경우 이메일 재발송 페이지로 이동하게 만들었습니다.
3. 방명록 / 신고기능
  • MySQL과 연결하여 방명록을 관리하도록 하였습니다.
  • 추천기능을 추가하여 MySQL에서 추천순으로 정렬할 수 있게 만들었습니다.
  • 추천은 한번만 누를 수 있게 userID를 MySQL에 같이 등록하도록 하였습니다.
  • 방명록은 5개씩 볼 수 있도록 MySQL를 이용해 페이징 기능을 만들었습니다.
  • MySQL을 이용해 제목과 내용을 선택해 검색할 수 있는 기능을 만들었습니다.
  • 신고 기능을 만들어 이용자들에게 메일을 받아 불적절한 내용들을 빠르게 관리 할 수 있도록 만들었습니다.
  • 글 등록 시 JavaScript 문법이 사용되지 않도록 replaceALL() 을 이용해 보안 기능을 만들어 보았습니다.
