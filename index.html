HOW TO STUDY

다양한 컴퓨터 언어 중 JAVA 라는 언어가 가장 많이 사용된다고 생각되어 강남의 한 학원에 가서 한번 배워보자 라는 마음으로 약 2달정도 배웠습니다. JAVA의 여러 문법들과 그것들을 이용해 여러 기능들을 만들어 보았습니다. 하지만 눈에 보이지 않는 기능들에 점점 제 스스로 이걸로 무엇을 만들 수 있을까라는 의구심이 생겼던 것 같습니다.

그래서 일단 눈에 보이는 Front 쪽 공부를 해보자는 마음으로 강의를 찾아보다 생활코딩이라는 무료 강의 사이트를 발견하고 그 곳에서 HTML과 CSS, JavaScript의 기초를 배울 수 있었습니다.
그 후 Front의 지식을 더욱 깊게 공부하고 싶다는 생각을 하였고, 애플코딩이라는 유료 강의사이트를 통해 조금 더 심화된 HTML, CSS, JavaScript를 공부하였습니다.
또 React라는 라이브러리를 공부하였습니다.

최근에는 Front와 Back 모두를 이용해 사이트를 만들어 보고싶다 라는 생각이 들어 JSP, JAVA, MySQL 을 이용해 로그인 기능과 게시판을 만들어 보았고, 나아가 SpringBoot와 Thymeleaf를 이용해 한층 더 짜임 있는 구성과 여러 다양한 기능들을 만들어 보았습니다.

  • MySQL에 연결하여 회원가입과 로그인 기능을 만들었습니다.
  • 로그인 상태에서는 로그아웃 버튼만 보이도록 만들었습니다.
  • 회원가입 페이지에서 아이디,비밀번호,이메일에는 정규식을 적용시켜보았고, 정규식에 부합하지 않은 경우 알림이 뜨게 만들었습니다.
